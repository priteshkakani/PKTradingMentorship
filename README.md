## Setup & Usage

1. **Open the website**: Simply open `index.html` in a web browser
2. **Local Development**: Use any local server (e.g., Live Server in VS Code)
3. **Deploy**: Upload all files to your web hosting service

## Customization

### Update Contact Information
- **WhatsApp**: Update the phone number in WhatsApp links (currently placeholder: 919876543210)
- **Email**: Already set to `priteshkakani@outlook.com`
- **UPI ID**: Already set to `priteshkakani@icici`

### Add Images
Replace the placeholder divs with actual images:
- Hero section: Professional photo
- Trading Proof: PnL screenshots and equity curve
- Testimonials: WhatsApp screenshots

### Colors
Colors are defined as CSS variables in `styles.css`:
- Royal Blue: `#1e3a8a`
- Gold: `#fbbf24`
- White: `#ffffff`
- Black: `#000000`
